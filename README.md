# drowning_study
a changelog for my drowning case study 
This should contain the jupyter notebook for my casestudy on drowning deaths in the US for 2023.
data files came from the NVSS mortality data for 2023 and the american community survey which is part of the census bureau.
There is a third data file from an unknown source. It is not integral for the analysis but was left in the notebook to show process
